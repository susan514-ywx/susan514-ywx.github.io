HTML基础

1.HTML标题通过< h1 >-< h6 >等标签进行定义的 。

  1. HTML段落是通过< p >标签进行定义的。
  2. HTML链接通过< a >标签定义。
    实例:
    < a href=”http://baidu.com “ > This is a link< /a>
    href=hypertext reference
    4.HTML图像通过< img>标签进行定义。
    实例：
    < img src=”批注 2020-01-18 161401.jpg” width=”104” height=”142”/>

HTML文档是由HTLM元素定义的

HTML元素

HTML元素是从开始标签（star tag/opening tag)到结束标签(end tag/closing tag)的所有代码。

嵌套的HTML元素

大多数HTML元素可以嵌套
HTML文档由嵌套的HTML元素构成。

空的HTML的元素

没有内容的HTML元素为空元素。空元素是在开始标签中关闭的。
< br>(定义换行）就是没有关闭标签的空元素。
但在XHTML,XML,及未来版本中的HTML中，所有元素必须关闭。
采用在开始标签中加斜杠/。即:< br/>

HTML提示：使用小标签

在未来（X)HTML强制使用小写。

HTML属性

属性为HTML元素提供附加信息。

属性实例：

  1. HTML链接
    < a href=”http://baidu.com “ >This is a link< /a>
  2. < h1>定义标题的开始
    < h1 align=”center”>拥有对齐方式的附加信息
  3. < body>d定义HTLM文档的主体
    < body bycolor=”yellow”>拥有关于背景颜色的附加信息。
  4. < table>定义HTLM表格
    < table border=”1”>拥有关于表格边框的附加信息。

    始终为属性值加引号

    双引号是最常用的，在某些情况下，可使用单引号
    例如：
    name=’Bill”hello world”‘

    HTLM提示：

  5. 使用小写属性
  6. 属性总是在HTLM元素的开始标签中规定
  7. 属性总是以名称——值对的形式出现。

    HTLM标题

在HTML文档中，标题很重要

HTML标题

通过< h1>-< h6>递减定义标题

注释：浏览器会自动地在标题前后添加空格

HTML水平线

< hr/>标签在HTLM页面中创建水平线
###HTML注释
实例：
<！–注释内容–>

HTML段落

HTML折行

在不产生新段落的情况下进行换行（新行），使用< br/>,虽然< br>与< br/>很相似，但在XHTLM，XML及未来的HTLM版本中只可使用< br/>。

HTML输出-有用的提示

所有连续的空格或空行都会算作一个空格，HTML代码中的所有连续的空行（换行)也被显示为一个空格。

HTML样式

style属性的作用：
提供了一种改变HTML元素的通用方法。

实例

1.背景颜色
< html>
< body style=”background-color:yellow”>
< h2 style=”background-color:red”>This is a heading< /h2>
< p style=”background-color:blue”>This is a paragraph< /p>
< /body>
< /html>
